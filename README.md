# LDN-Scraper
Web scraper to retrieve the raw text from https://www.lagrangenews.com

In the main method of LDNScraper.py, pass in a link such as "https://www.lagrangenews.com/2023/06/02/troup-high-school-announces-new-principal/"
and the code will create a txt file with the title as the name of the txt file and the text of the story as the contents. Example output:
